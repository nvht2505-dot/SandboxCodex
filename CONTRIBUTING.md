# Contributing

Cảm ơn bạn đã quan tâm đến Sandbox CodeX.

## Quy trình đóng góp

1. Tạo issue mô tả tính năng hoặc lỗi.
2. Tạo branch theo định dạng `feature/<ten-tinh-nang>` hoặc `fix/<ten-loi>`.
3. Cập nhật tài liệu và kiểm thử liên quan.
4. Mở pull request với mô tả rõ ràng.

## Chuẩn chất lượng

- Code rõ ràng, dễ đọc và có cấu trúc module.
- Không commit secrets hoặc token.
- Ưu tiên kiểm thử tự động cho logic quan trọng.
